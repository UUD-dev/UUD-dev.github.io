Plugin Setup

Enable WebSocket Server

Enable the WebSocket Server in Streamer.bot using the following path: 

Servers / Clients
  └─ WebSocket Server

WebSocket Configuration

These settings should not need to be changed. They are the default Streamer.bot WebSocket settings: 

Auto Start:       On
Address:          127.0.0.1
Port:             8080
Endpoint:         /
Authentication:   Disabled

Using the Plugin with OBS

After completing the steps below, a window should appear displaying both your Twitch and YouTube chats.

  1. Open OBS
  2. At the top menu, click Docks
  3. Select Docks > Custom Browser Dock
  4. Under Dock Name, enter a descriptive name (example: UUDChatBot)
  5. Under URL, enter the following:

Dock URL

https://uud-dev.github.io/OBS_dock/index.html

Final Step

Everything should now be set up and ready to use. You should see a "connected" message appear in the dock.
